# ORS Holding Page – GitHub Pages

One-page holding website for **Open Research Shop (ORS)** — a UK startup tokenising and crowdfunding academic research with GBP-native infrastructure.

- Live: https://openresearch.shop (custom domain; also https://openresearchshopuk.github.io/openresearchshop-holding)
- Stack: single `index.html` (embedded CSS + vanilla JS), Poppins via Google Fonts, no build step
- Deployed via GitHub Pages (main branch, `/` root)
